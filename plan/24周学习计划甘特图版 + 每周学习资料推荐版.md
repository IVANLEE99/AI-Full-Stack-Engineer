## 二、每周学习资料推荐版

### 第1周：Go语法入门
**学习目标**
- 建立 Go 强类型与显式错误处理思维

**推荐资料**
- Go 官方 Tour
- Go by Example
- 《Go语言圣经》前 1-5 章

**实战任务**
- 用 Go 重写一个你熟悉的前端工具函数模块
- 完成 Todo API CRUD

**验收标准**
- 能解释 `if err != nil`
- 能写基础 REST API
- 能读懂 AI 生成的 Go 代码并手改

---

### 第2周：Gin Web基础
**学习目标**
- 能用 Gin 写标准 API

**推荐资料**
- Gin 官方文档
- validator 官方示例
- JWT 最佳实践文章

**实战任务**
- 用户注册
- 用户登录
- JWT 鉴权
- 用户信息接口

**验收标准**
- 能解释中间件执行顺序
- 注册/登录/鉴权链路完整

---

### 第3周：数据库与ORM
**学习目标**
- 打通 Go + 数据库 + 缓存

**推荐资料**
- GORM 官方文档
- PostgreSQL 或 MySQL 基础教程
- Redis 官方入门文档

**实战任务**
- 商品、分类、库存模块
- 商品 CRUD
- 商品详情缓存

**验收标准**
- 会建模
- 能完成 Go + DB + Redis 联调
- 能讲清数据流转

---

### 第4周：项目分层架构
**学习目标**
- 把 demo 升级为工程项目

**推荐资料**
- Go Project Layout
- Clean Architecture 入门文章
- Wire / Dig 依赖注入资料

**实战任务**
- 重构项目目录
- 建立 CMS Admin 后端骨架
- 统一错误返回

**验收标准**
- 项目具备清晰分层
- 后续项目可以复用该骨架

---

### 第5周：JWT与权限控制
**学习目标**
- 理解企业系统基础权限模型

**推荐资料**
- JWT 最佳实践
- RBAC 基础文章
- Gin 鉴权中间件示例

**实战任务**
- 登录签发 Token
- 受保护路由
- 简单角色权限控制

**验收标准**
- 未登录无法访问受保护接口
- 不同角色可见资源不同

---

### 第6周：事务与核心业务建模
**学习目标**
- 具备真实业务建模能力

**推荐资料**

- DDD 入门文章
- 高并发库存扣减案例

**实战任务**

- 扣库存
- 防超卖
- 订单状态流转

**验收标准**
- 核心链路正确
- 能解释事务和锁的作用

---

### 第7周：Go并发基础
**学习目标**

**验收标准**

### 第8周：第一阶段综合项目

**实战任务**

### 第9周：Redis生产实战
**学习目标**
- 学会做抗压接口

**推荐资料**

**实战任务**

**验收标准**

---

### 第10周：并发进阶与Worker Pool
**学习目标**
- 可控地处理大批量任务

**推荐资料**

**实战任务**

**验收标准**
- 不滥开 goroutine
- 有超时和失败处理

## 三、每月项目目标

| 月份 | 必做项目 | 目标 |
|---|---|---|
| 第1个月 | CMS Admin / Todo API / 商品后台 | 建立 Go Web 基础能力 |
| 第2个月 | 订单系统 / 库存系统 | 掌握事务、并发、缓存 |
| 第3个月 | Docker 化后台系统 + 自动化测试 | 具备生产级工程意识 |
| 第4个月 | LLM Gateway / 流式聊天接口 | 进入 AI 服务开发 |
| 第5个月 | 企业知识库 / FAQ Bot / Tool Agent | 掌握 RAG 与 Tool Calling |
| 第6个月 | AI 代码审查 Agent / AI Native Platform | 完成毕业项目 |
